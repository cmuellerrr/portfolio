Overview

As part of an independent study, I took on the task of trying to allow direct I/O from a web browser using native JavaScript – no plugins. Plugins are entirely unnecessary and this project was trying to put a dent in that model. The goal was to allow developers to to handle complex I/O like webcam input using simple JavaScript objects without having to force users to download a Flash plugin. I ended up choosing Firefox as the browser to modify because of its maturity and wide user base.

Implementation

I developed a proof of concept of this by creating a native JavaScript object for the PlayStation3 Dualshock 3 controller. In order to do this I had to first create a simple Linux driver for the controller which could read what buttons were pressed (and how hard). That driver was then hooked up to the C implementation of the JavaScript object to pass the data up to the web browser. In the end, the browser was able to see all input from the controller using one object. Admittedly, the project didn’t get as far as I had hoped, but it was very close. Ideally, this concept could have been abstracted further to allow for easy extension in addition to a more useful proof of concept like being able to navigate a web page using the controller.
